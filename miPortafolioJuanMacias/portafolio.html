Portafolio

  • Conversor de unidades de longitud

    Este programa en lenguaje Python, permite a través de una función realizar la conversión de unidades de medida de longitud, tanto el valor de origen como destino y mostrando el valor ya convertido.

  • Programa para validacion de cedula ecuatoriana

    Este programa contiene una función que valida un numero de cedula y devuelve un booleano que indica si la cedula es valida o no en los casos que el numero este incompleto o parametros diferentes

  • Programa para validad si un numero de 4 digitos par o impar

    Este programa permita valida el ingreso de un numero entero positivo de 4 cifras, luego de cumplir la condición determina si es par o impar, en caso de ser par muestra dicho numero invertido y si el numero es impar mostrar la suma de sus dígitos.

  • Algoritmo de Dijkstra: camino mas corto entre ciudades de Argentina

    Este programa realiza el calculo del camino mas corto entre ciudades de Argentina utilizando el algoritmo de Dijkstra y librerias python como networkx, matplotlib.pyplot y numpy.

  • Programa para ingresar 2 valores y mostrar la ubicacion en un plano cartesiano

    Este programa permite ingresar dos valores numéricos menores a 4 cifras, los mismos que corresponden a las coordenadas X y Y. Hace la validacion que los valores ingresados sean numéricos, y determina en qué lugar del plano cartesiano se encuentra el punto en algunos de los cuadrante.
